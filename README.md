# ShopSphere 🛒

A modern Amazon-inspired ecommerce web application built using Next.js, TypeScript and Tailwind CSS.

---

## ✨ Features

- Modern Amazon-style UI
- Product listing page
- Product details page
- Shopping cart drawer
- Wishlist functionality
- Dark mode support
- Category filters
- Price filtering
- Responsive design
- Pagination system
- Add to cart functionality

---

## 🛠 Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Redux Toolkit
- React Icons

---

## 📸 Preview

An advanced ecommerce frontend inspired by Amazon with smooth UI and responsive layouts.

---

## 🚀 Getting Started

Run the development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
